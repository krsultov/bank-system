# Банков софтуер

Конзолно приложение на C за прости банкови операции: регистрация, логин,
депозит, теглене и трансфер между сметки.

## Компилиране и стартиране

    make
    ./bank

## Структура

    include/   заглавни файлове (структури и декларации)
    src/       изходен код
    *.dat      данни в двоичен формат (потребители, сметки, транзакции)

Данните се пазят в три файла и се зареждат при стартиране. Ако файловете ги
няма, програмата започва с празни колекции. След всяка промяна съответният
файл се презаписва.

Трансферите не се изпълняват веднага — влизат в опашка и се извършват при
избор на "Обработка на транзакции".

## Разпределение на работата

- Регистрация и логин (хеширани пароли) — Ева
- Сметки и операции (депозит, теглене, трансфер, обработка) — Крум
